# FantasyLedger (Capstone 1)

This is a financial management application with a game inspired fantasy-themed interface for tracking payments, deposits, vendors, and dates.

## Description

This is a capstone project. Feedback and suggestions are welcome! FantasyLedger is a personal finance tracker that gamifies money management through a fantasy-themed user interface. Users can manage their financial data while enjoying an immersive medieval/fantasy experience with customized menus and visual elements.

## Features

- **Income Tracking**: Record and categorize various income sources
- **Expense Management**: Track spending across different categories
- **Budget Planning**: Set and monitor budget goals
- **Data Persistence**: Save financial data to files for future sessions
- **Fantasy-Themed UI**: Immersive medieval/fantasy interface design
- **Interactive Menus**: User-friendly navigation through all features
